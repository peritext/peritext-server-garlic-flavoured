Contextualiseur table

Le Contextualiseur de type table affiche une ressource de type table (exemple: tire d'un fichier csv) sous la forme d'un tableau traditionnel.

En version codex, il produit un tableau simple.

En version web, il produit un tableau interactif (pagination, recherche, ...)
